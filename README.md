Book_Homestay项目是一个房东民宿出租与用户预定民宿软件的后台开发
# 该项目目前实现业务功能包括：
1. 创建民宿订单，用户订单列表，用户订单明细
2. 完成三方支付方式：vx支付
3. 获取民宿信息，民宿boss信息，民宿订单信息，猜你喜欢民宿，最佳房东推荐
4. 用户登入注册（注册与登入支持三方登入：vx登入），获取用户信息
# 项目目前的功能包括：
订单服务,支付服务,用户服务，民宿服务，网关，鉴权，错误处理，日志收集，服务监控，链路追踪
